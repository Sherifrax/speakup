# speakup

A front-end project for TBMS 2.1 SpeakUp UI built with Vite + React + TypeScript and Tailwind.

## Structure
- `tbms-2-1-new/`: current app source (`src/`), config, and build outputs
- `tbms-2-1-old/`: previous version kept for reference

## Development
Install dependencies and start the dev server:
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
